# 创建桌面快捷方式指南

## Windows系统

### 方法1：创建批处理快捷方式
1. 复制 `start.bat` 文件到桌面
2. 右键点击 → "发送到" → "桌面快捷方式"
3. 双击快捷方式即可启动

### 方法2：使用Python启动器
1. 复制 `快捷启动.py` 文件到桌面
2. 双击运行，会自动启动并打开浏览器

## macOS系统

### 方法1：创建应用程序快捷方式
1. 打开"自动操作"（Automator）
2. 选择"应用程序"
3. 添加"运行Shell脚本"操作
4. 输入：
   ```bash
   cd "/Users/a1234/Claude/test/acu_assistant"
   python3 app.py
   ```
5. 保存为"智能针灸助手"
6. 双击即可启动

### 方法2：使用终端别名
1. 打开终端
2. 输入：
   ```bash
   echo 'alias acu_assistant="cd /Users/a1234/Claude/test/acu_assistant && python3 app.py"' >> ~/.zshrc
   source ~/.zshrc
   ```
3. 以后只需在终端输入 `acu_assistant` 即可启动

## Linux系统

### 方法1：创建桌面文件
1. 在桌面创建 `smart-acupuncture.desktop` 文件
2. 内容如下：
   ```ini
   [Desktop Entry]
   Version=1.0
   Type=Application
   Name=智能针灸助手
   Comment=智能针灸辨证选穴助手
   Exec=/path/to/your/python3 /Users/a1234/Claude/test/acu_assistant/app.py
   Icon=applications-office
   Terminal=true
   Categories=Office;
   ```
3. 给文件添加执行权限：`chmod +x smart-acupuncture.desktop`

### 方法2：使用启动器
1. 复制 `start.sh` 到桌面
2. 右键属性 → 权限 → 勾选"允许作为程序执行文件"
3. 双击运行

## 浏览器快速访问

将以下HTML文件保存到桌面，双击即可一键启动并打开网页：

### 1. 创建快捷方式
创建一个包含以下内容的HTML文件（保存为 `launch.html`）：

```html
<!DOCTYPE html>
<html>
<head>
    <title>智能针灸助手</title>
    <script>
        window.onload = function() {
            // 提示用户选择启动方式
            if (confirm("是否启动智能针灸助手？")) {
                window.location.href = "file:///Users/a1234/Claude/test/acu_assistant/快捷启动.py";
            }
        }
    </script>
</head>
<body>
    <h1>智能针灸辨证选穴助手</h1>
    <p>点击确定启动程序</p>
</body>
</html>
```

### 2. 使用浏览器书签
在浏览器中创建书签，地址指向：
```
file:///Users/a1234/Claude/test/acu_assistant/launch.html
```

## 快速启动命令

### Windows
```batch
# 双击或在命令行运行
start.bat
```

### macOS/Linux
```bash
# 双击或在终端运行
./start.sh
```

### Python启动器（推荐）
```bash
# 双击或在终端运行
python3 快捷启动.py
```

## 一键关闭

1. **Windows**：在命令行窗口按 `Ctrl+C`
2. **Mac/Linux**：在终端按 `Ctrl+C`
3. **直接关闭**：关闭命令行/终端窗口

## 注意事项

1. 确保Python 3.6+已安装
2. 首次运行会自动安装依赖
3. 如果端口8000被占用，请修改 `app.py` 中的端口号
4. 建议将整个项目文件夹备份，以防数据丢失
