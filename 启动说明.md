# 智能针灸辨证选穴助手 - 快速启动指南

## 🚀 启动方式

### 方式一：双击启动（推荐）

1. **Windows用户**
   - 双击 `start.bat` 文件
   - 程序会自动启动并提示服务器启动成功

2. **Mac/Linux用户**
   - 双击 `start.sh` 文件
   - 或在终端中运行：`./start.sh`

### 方式二：使用Python启动器

1. 双击 `快捷启动.py` 文件
2. 选择是否自动打开浏览器
3. 程序会自动检查依赖并启动

### 方式三：命令行启动

```bash
# Windows
python app.py

# Mac/Linux
python3 app.py
```

## 🌐 访问方式

启动成功后，在浏览器中访问：
- http://localhost:8000

## 📂 文件说明

| 文件名 | 作用 | 说明 |
|--------|------|------|
| `start.bat` | Windows启动脚本 | 双击即可启动程序 |
| `start.sh` | Mac/Linux启动脚本 | 双击或终端运行 |
| `快捷启动.py` | Python启动器 | 带自动打开浏览器功能 |
| `launch.html` | HTML启动页面 | 可以保存到桌面，点击启动 |
| `app.py` | 主程序 | FastAPI后端服务 |

## ⚡ 自动关闭程序

1. 在终端窗口中按 `Ctrl+C`
2. 或直接关闭终端窗口

## 🔧 故障排除

### 1. 提示"python不是内部或外部命令"
- 安装Python：https://www.python.org/downloads/
- 或在命令行中使用完整路径：`C:\Python39\python app.py`

### 2. 提示"端口被占用"
- 修改 `app.py` 中的端口号（第356行）
- 将 `port=8000` 改为其他端口，如 `port=8080`

### 3. 浏览器无法访问
- 确保程序正在运行（终端显示服务器启动信息）
- 检查防火墙设置
- 尝试访问 http://127.0.0.1:8000

## 💡 使用技巧

1. **创建桌面快捷方式**
   - 将 `start.bat`（Windows）或 `start.sh`（Mac）复制到桌面
   - 双击即可启动

2. **设置开机自启动**
   - Windows：将 `start.bat` 添加到启动文件夹
   - Mac/Linux：将启动脚本添加到系统偏好设置的用户登录项

3. **快速访问**
   - 将 `launch.html` 保存到桌面
   - 双击即可启动并打开网页

## 📱 移动设备访问

在手机或平板上也能使用：
1. 确保电脑和手机在同一WiFi网络
2. 查看电脑的IP地址（在命令行运行 `ipconfig` 或 `ifconfig`）
3. 在手机浏览器中访问：`http://电脑IP地址:8000`
