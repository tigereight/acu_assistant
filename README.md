# 智能针灸辨证选穴助手 (AcuPredictor)

基于中医针灸理论的智能选穴 Web 应用，用户输入症状后，程序会根据辨证逻辑推荐核心穴位及其配伍方案。

## 功能特点

- **智能辨证**：根据症状和辨证条件，自动匹配中医辨证类型
- **穴位推荐**：提供主穴、配穴及其详细定位信息
- **针刺指导**：包含针刺注意事项和操作要点
- **模糊搜索**：支持症状关键词模糊匹配

## 快速开始

### 1. 安装依赖

```bash
cd acu_assistant
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

或者使用 uvicorn：

```bash
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

### 3. 访问应用

打开浏览器访问：http://localhost:8000

## API 接口

### 获取所有病症

```
GET /api/symptoms
```

### 搜索穴位推荐

```
POST /api/search
Content-Type: application/json

{
    "query": "头痛",
    "conditions": ["受寒", "怕冷"]
}
```

### 获取穴位详情

```
GET /api/acupoint/{name}
```

## 辨证逻辑示例

用户输入症状"头痛"，并选择辨证条件"受寒"：

1. 系统匹配到"头痛"病症
2. 根据条件"受寒"匹配到"风寒"辨证类型
3. 返回推荐穴位：
   - **主穴**：风池、合谷、太阳
   - **配穴**：列缺、外关
4. 提供辨证逻辑：风寒袭络，治以祛风散寒、通络止痛
